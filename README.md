## 🌐 How to Deploy on GitHub Pages

1. Create a repository named `portfolio`

2. Upload:

   * `index.html`
   * `style.css`
   * `script.js`

3. Go to:

   * **Settings → Pages**

4. Under Source:

   * Select **Deploy from branch**
   * Choose `main` branch
   * Select `/ (root)`
   * Click **Save**

5. Your site will be live at:

```
https://yourusername.github.io/portfolio/
```

---

## ❗ Important Notes

* File names must be exact:

  * `index.html`
  * `style.css`
  * `script.js`
* Do NOT put files inside extra folders
* Paths must match exactly (case-sensitive)
